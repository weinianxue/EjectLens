# Manual Steps for v0.1.0 Release

These steps require a human to complete because the current environment
does not have access to the GitHub Releases API or `gh` CLI.

## 1. Create GitHub Release

1. Go to https://github.com/weinianxue/EjectLens/releases/new
2. Set tag to: `v0.1.0` (create new tag on publish)
3. Set release title to: `EjectLens v0.1.0`
4. Copy release notes from `docs/release-notes/v0.1.0.md`
5. Attach the zip file: `EjectLens-v0.1.0-win-x64.zip`
   (This file is in the repository root, generated by `dotnet publish`)

## 2. Verify CI

After pushing, check the Actions tab to confirm the CI workflow runs:
https://github.com/weinianxue/EjectLens/actions

## 3. Add screenshots (optional)

On a Windows machine with a USB drive connected:

1. Build and run EjectLens.
2. Take screenshots of the main window with events displayed.
3. Save to `docs/screenshots/main-window.png`
4. Optionally capture additional states (no events, report panel).
5. Commit the images and update README references.
