# EjectLens v0.1.0

Initial public preview.

## What's included

- Lists removable drives with volume details (label, file system, capacity).
- Reads recent Microsoft-Windows-Kernel-PnP Event ID 225 entries from the System event log.
- Shows processes that may be blocking safe device ejection.
- Color-codes events as Matched, Possibly related, or Unmatched by drive.
- Lightweight Restart Manager based file lock scanner for selected folders.
- Copy or save diagnostic reports as plain text.
- One-click Task Manager access.

## Safety

EjectLens is read-only by default. It does not terminate processes, force-eject devices, modify the registry, or connect to the network.

## Requirements

- Windows 10 or Windows 11
- .NET 8 Desktop Runtime

## Known limitations

- Event ID 225 must be present in the System event log for the tool to identify blocking processes.
- Reading the System event log may require administrator privileges on some systems.
- The file lock scanner is capped at 300 files per folder to avoid performance issues.
- Volume-to-drive matching uses heuristic comparison and may not always be exact.
