# Screenshots

Screenshots will be added after capturing the application on a Windows 10
or Windows 11 desktop with a removable drive connected.

This directory is currently empty in the source repository. Real screenshots
should be captured from a running instance of EjectLens. Do not use mock or
fabricated screenshots — they would be misleading for a diagnostic tool.

To add screenshots:

1. Build and run EjectLens on a Windows machine.
2. Capture the main window showing events for a connected removable drive.
3. Save to `docs/screenshots/main-window.png`.
4. Optionally capture:
   - `report-panel.png` — the detail panel showing event information
   - `no-events.png` — the application state when no events are found
   - `settings.png` — the Settings dialog
5. Update `README.md` to reference the images.
